# Student Performance Analysis

## Objective
Analyze student performance using Python.

## Dataset
Kaggle student-por dataset

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Total & Average calculation (G1, G2, G3)
- Gender analysis
- Study time analysis
- Data visualization

## How to run
pip install pandas matplotlib seaborn  
python analysis.py
